# Evaluation Plan

Energy Trading Lab should be evaluated as an AI Agent application and as a research-simulation scaffold.

## Agent engineering metrics

- Strategy classification accuracy across RL/MARL, optimization, auction, game theory, and rule-based methods.
- Structured extraction completeness for state, action, reward/objective, baseline, metrics, and grid constraints.
- Tool-call success rate across simulation, grid validation, and report generation.
- Failure recovery rate after one repair attempt.
- Average run latency and token/cost estimates after LLM integration.

## Research scaffold metrics

- Reproduction package completeness: model spec, strategy spec, gaps, config, metrics, report.
- IEEE 33/69 run success rate.
- Baseline comparability across traditional and RL strategies.
- Explicit separation of paper facts, Agent assumptions, and local simulation results.
